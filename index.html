使用说明

查看完整说明 →
⬆️

上传图片

点击上传区或拖拽图片到页面，支持批量选择（最多 10 张）。也可按 Ctrl+V 直接粘贴截图。

✏️

粘贴 SVG 代码

切换到「粘贴代码」Tab，直接粘贴 SVG 源码或 Base64 格式的 SVG Data URL。

⚙️

调整参数

JPG/BMP 可设置最大宽度与压缩质量；PNG 仅支持缩放；SVG 支持双引号模式。

📋

复制结果

选择输出格式（纯 Base64 / CSS / HTML），点击「复制」按钮即可使用。

🕘

转换历史

每次转换自动保存到本地 IndexedDB。点击历史卡片上的配置按钮可一键还原参数并重新转换。

🔒

完全本地

所有转换在浏览器内完成，图片数据不会上传到任何服务器，安全可靠。
